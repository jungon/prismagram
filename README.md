# prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] Log in
- [ ] See the feed
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] See user profile
- [ ] Follow / Unfollow user
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Create account
- [ ] Upload a photo
- [] Edit the photo (Delete)
