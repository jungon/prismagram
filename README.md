- [ X ] Create account
- [ ] Request Secret
- [ ] Confirm Secret (Login)
- [ ] Log in
- [ ] See the feed
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] See user profile
- [ ] Follow / Unfollow user
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
